Checkaroo Library Documentation

Documentation for the core shared logic and utility libraries in @/common/*.

Database & Storage

dbutils Source: src/common/dbutils.ts

Abstracted database operations using mysql2 connection pooling.

select(query: SelectQuery)

Standardized SELECT queries with support for joins, where clauses, sorting, and limits.

insert(query: InsertQuery)

Handles data insertion into specific tables.

update(query: UpdateQuery)

Handles partial or full record updates.

deleteRec(query: DeleteQuery)

Removes records from the database.

cookieServer Source: src/common/cookieServer.ts

Server-side cookie management using Next.js headers/cookies.

writeCookie(name, data, options)

Sets a cookie (defaults to 7-hour expiration).

readCookie(name)

Retrieves a cookie value on the server.

deleteCookie(name)

Clears a cookie by setting an immediate expiration.

Security & Auth

session Source: src/common/session.ts

Business logic for session validation and user authentication.

checkValidSession(hash)

Validates a session hash against the database and expiration time.

headersLegit(request, routes)

Security middleware checking referer and larva integrity headers.

validateUser(user, passHash)

Checks credentials against the User table.

crypt Source: src/common/crypt.ts

AES Encryption services using crypto-js.

encrypt(data) / decrypt(data)

Base AES encryption with URI encoding for safe transmission.

superEncrypt(session)

A specialized layered encryption used for secure session identifiers.

General Utilities

common Source: src/common/common.ts

Shared helper functions used across the frontend and backend.

formatDouble(amount)

Standardizes currency formatting (fixed to 2 decimals).

convertToMySQLDate(date) / convertToNiceDate(date)

Date conversion helpers using moment.js.

decodeHtmlEntities(text)

Parses HTML strings to retrieve raw text content.

logs Source: src/common/logs.ts

Standardized logging utility that writes to stdout.

writelog(msg, premsg, level)

Outputs formatted logs based on environment and severity levels.
