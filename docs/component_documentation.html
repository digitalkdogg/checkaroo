Checkaroo Component Documentation

Comprehensive guide to the reusable UI components used in the Checkaroo application.

Navigation & Layout

Leftside Source: src/app/components/Leftside.tsx

The primary navigation sidebar. It handles responsive layouts (sidebar on desktop, hamburger menu on mobile).

Props: 
interface Props {
  enable: boolean; // Enables Balance and Menu links
  session: string;
}
Menu Source: src/app/components/Menu.tsx

A standard navigation link item used within the Leftside component.

Props: 
interface Props {
    enable: boolean;
    link: string;
    text: string;
}
Addbutton Source: src/app/components/Addbutton.tsx

A floating-style action button that redirects the user to a specific URL.

Props: 
interface Props {
  url: string;
}

Data Display

Balance Source: src/app/components/Balance.tsx

Displays and animates the account balance.

Props: 
interface Props {
  enable?: boolean;
  session: string;
}
Dashboard Source: src/app/components/Dashboard.tsx

Server-side component that orchestrates the main transaction list.

Props: 
interface Props {
  session: string;
}
RelatedItems Source: src/app/components/RelatedItems.tsx

Displays a summary of transactions related to a specific ID.

Props: 
interface Props {
  session: string;
  id: string;
  api: string;
}
Transrow Source: src/app/components/Transrow.tsx

Renders a single clickable transaction row.

Props: 
interface Props {
    transId: string;
    date: string;
    companyName: string;
    amount: number;
    categoryName: string;
}

interface Props {

Checkaroo-styled date selection component.

interface Props {

}
CustomButton Source: src/app/components/Button.tsx

Form submission button with automatic payload gathering.

Props: 
interface props {
    id: string;
    className?: string;
    text: string;
    session: string;
    url: string;
    payload: Array; // IDs of form fields to include
    callBack: (data: {msg:string, status: string}) => void;
}
