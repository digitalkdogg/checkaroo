Checkaroo API Documentation

Comprehensive guide to the Checkaroo internal API endpoints.

Balance Management

POST /api/balance Session Required
Source: src/app/api/balance/route.ts

Retrieves the current account balance.

Success Response: 
[{ "balance": 1250.50 }]
POST /api/balance/update Session Required
Source: src/app/api/balance/update/route.ts

Updates balance by a specific value.

Request Data: 
{ "value": "50.00" }
Success Response: 
{
  "status": "success",
  "balance": 1,
  "newbalance": 1200.50,
  "oldbalance": 1250.50
}

Category Management

POST /api/categories Session Required
Source: src/app/api/categories/route.ts

Lists all categories.

Success Response: 
[
  { "category_id": 1, "category_name": "Food", "account_id": 10 },
  { "category_id": 2, "category_name": "Rent", "account_id": 10 }
]
POST /api/categories/add Session Required
Source: src/app/api/categories/add/route.ts

Creates a new category.

Success Response: 
{ "status": "success", "message": "category added successfully" }
POST /api/categories/delete Session Required
Source: src/app/api/categories/delete/route.ts

Removes a category (if not linked to transactions).

Success Response: 
{ "message": "Category Removed Successfully", "status": "success" }
Error Response: 
{ "message": "Category is tied to one or more transactions", "status": "error" }

Transactions

POST /api/transaction/add Session Required
Source: src/app/api/transaction/add/route.ts

Records a new transaction.

Success Response: 
{
  "status": "success",
  "transid": "8f2a...",
  "amount": "50.00",
  "message": "Transaction added successfully"
}
POST /api/transaction/suggested-category Session Required
Source: src/app/api/transaction/suggested-category/route.ts

Suggests a category based on company history.

Success Response: 
{
  "status": "success",
  "category_id": [{ "category_id": 5 }]
}
POST /api/dashboard Session Required
Source: src/app/api/dashboard/route.ts

Retrieves transaction list for the dashboard.

Success Response: 
{
  "results": [
    { "trans_id": "...", "date": "...", "amount": "...", "company_name": "...", "category_name": "..." }
  ]
}

Authentication

POST /api/login/auth Public
Source: src/app/api/login/auth/route.ts

Authenticates user and creates session.

Success Response: 
{ "status": "success", "message": "Login Success! Redirecting now" }
Error Response: 
{ "status": false, "message": "Bad username and password..." }
